# Component 4: Garbage Detection

## Overview
This component detects and tracks garbage/debris in the swimming pool for automatic cleaning notifications.

## Features
- Real-time garbage detection
- Location tracking
- Cleaning alerts
- Contamination history

## Directory Structure
- `datasets/` - Images of garbage and debris
- `models/` - Object detection models
- `scripts/` - Detection and tracking scripts

## Dependencies
See `requirements.txt` in the root directory
