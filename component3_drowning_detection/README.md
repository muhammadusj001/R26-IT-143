# Component 3: Drowning Detection

## Overview
This component provides real-time drowning detection using advanced computer vision and pose estimation techniques.

## Features
- Real-time pose detection
- Drowning risk assessment
- Emergency alerts
- Multi-person tracking

## Directory Structure
- `datasets/` - Training data for pose detection
- `models/` - Pose estimation and detection models
- `scripts/` - Detection and analysis scripts

## Dependencies
See `requirements.txt` in the root directory
