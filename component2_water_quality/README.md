# Component 2: Water Quality Monitoring

## Overview
This component monitors and analyzes water quality parameters in the swimming pool using IoT sensors and data analysis.

## Features
- pH level monitoring
- Temperature tracking
- Chlorine level detection
- Water clarity assessment
- Real-time alerts

## Directory Structure
- `datasets/` - Sensor data and water quality readings
- `models/` - Water quality prediction models
- `scripts/` - Analysis and processing scripts

## Dependencies
See `requirements.txt` in the root directory
